Prompt 1 — Prompt para Render de Fachada

Objetivo:

Gerar um prompt fotorrealista de arquitetura externa para o Imagen 3.

Instrução (Pronta para usar):
Você é o ArquiMatch PRO. Elabore um prompt em inglês técnico de alta precisão fotográfica para o Imagen 3 gerar uma fachada residencial de estilo [ESTILO], com acabamentos em [MATERIAIS]. A cena deve estar ambientada sob iluminação de [HORARIO] em um dia [CLIMA], utilizando câmera com lente de [LENTE] e foco focado no detalhe geométrico da entrada.
Campos para substituir: [ESTILO] (ex: minimalista contemporâneo), [MATERIAIS] (ex: concreto aparente ripado, madeira cumaru e esquadrias pretas), [HORARIO] (ex: golden hour / entardecer), [CLIMA] (ex: ligeiramente úmido com reflexos na calçada), [LENTE] (ex: 24mm tilt-shift)

Prompt 2 — Prompt para Render de Interior

Objetivo:

Gerar um prompt fotorrealista de sala ou quarto para IA.

Instrução (Pronta para usar):
Você é o ArquiMatch PRO. Elabore um prompt técnico e artístico para renderizar uma sala de estar com pé-direito duplo integrada com cozinha gourmet. O estilo de design de interiores deve ser [ESTILO_INTERIOR], com destaque para a textura de [TEXTURA_DESTAQUE] e iluminação proveniente de [ILUMINACAO_FONTE].
Campos para substituir: [ESTILO_INTERIOR] (ex: escandinavo moderno com toques brasileiros), [TEXTURA_DESTAQUE] (ex: sofá de linho cru, tapete de fibra natural e painel de freijó), [ILUMINACAO_FONTE] (ex: grandes aberturas de vidro com luz difusa de claraboia)

Prompt 3 — Memorial Conceitual (Copywriting)

Objetivo:

Escrever a narrativa conceitual e de partido para encantar o cliente.

Instrução (Pronta para usar):
Você é o ArquiMatch PRO. Escreva um memorial descritivo conceitual de apresentação para o projeto de uma [TIPO_PROJETO] com área de [AREA] m². O conceito de projeto é focado em [CONCEITO_FOCO] e integra soluções de sustentabilidade como [SUSTENTABILIDADE]. Adote um tom poético, elegante e técnico simultaneamente.
Campos para substituir: [TIPO_PROJETO] (ex: residência unifamiliar serrana), [AREA] (ex: 280), [CONCEITO_FOCO] (ex: integração interna-externa e sensação de acolhimento), [SUSTENTABILIDADE] (ex: ventilação cruzada natural, aproveitamento de água de chuva e energia fotovoltaica)

Prompt 4 — Roteiro de Tour Virtual (Vídeo)

Objetivo:

Roteirizar cenas para geração de vídeo conceitual.

Instrução (Pronta para usar):
Você é o ArquiMatch PRO. Crie o roteiro detalhado de cenas (storyboard) para um vídeo conceitual de [DURACAO] segundos apresentando a transição do [CENA_ORIGEM] para a [CENA_DESTINO], ideal para gerar um clipe de animação no Google Veo 3.1, descrevendo movimentos de câmera lentos e cinematic.
Campos para substituir: [DURACAO] (ex: 15), [CENA_ORIGEM] (ex: jardim de entrada da casa sob a luz da manhã), [CENA_DESTINO] (ex: área social interna com transição de cortinas movidas a vento suave)

Perguntas Frequentes

Dúvidas comuns sobre o ArquiMatch PRO

O ArquiMatch PRO gera modelos 3D editáveis (BIM/CAD)? +
Não. O assistente cria mockups conceituais de imagem/vídeo e orientações de visualização e design, servindo para aprovação de conceito rápido. A modelagem 3D técnica continua exigindo softwares como Revit, SketchUp ou Archicad.
Como funciona a geração de vídeo com o Veo 3.1? +
O agente ajuda a estruturar a narrativa técnica e a descrever a trajetória da câmera, os elementos em movimento e a luz para que você use na ferramenta de geração de vídeos (Veo) do ecossistema Google.
Posso usar os memoriais gerados para fins comerciais? +
Sim, os memoriais descritivos e textos conceituais são de sua propriedade intelectual e podem ser usados comercialmente para apresentação de projetos a clientes.
Quais são os prompts mais eficientes para renders internos? +
O agente fornece prompts com tags específicas como iluminação volumétrica, materiais PBR (wood grain, brushed steel), especificações de lente (35mm f/1.8) e estilo de composição visual.
É necessário ter uma licença premium do Imagen 3? +
Não. O Imagen 3 e o Gemini estão disponíveis gratuitamente nas plataformas oficiais do Google, embora as versões pagas possam oferecer processamento acelerado.
Consigo simular reformas (retrofit) com fotos existentes? +
Sim. Na versão multimodal do Gemini, você carrega a foto do imóvel existente e utiliza o prompt do ArquiMatch PRO especificando quais elementos manter e quais reformar.
O agente ajuda a definir a paleta de cores do projeto? +
Com certeza. Ele sugere paletas de cores de acordo com teorias de psicologia ambiental, estilo arquitetônico e as tendências de design de interiores vigentes.
Quantas imagens posso gerar por dia? +
A geração de imagem é feita na plataforma do Gemini, sujeita aos limites de uso diário definidos pelas políticas de uso do ecossistema do Google.
