O Que é o EngMatchFree?

O EngMatchFree é o agente central do ecossistema ObraMatch, funcionando como um consultor de engenharia civil completo. Ele integra as normas técnicas brasileiras (ABNT) com ferramentas avançadas de IA do Google, oferecendo suporte técnico multimodal para projetos de construção.

Como Funciona?

O agente opera em quatro pilares principais:

1. Garantia de Conformidade Normativa

  • Analisa projetos comparando com NBR 15575 (Desempenho) e NBR 6118 (Estruturas)
  • Verifica atualizações de normas em tempo real via Google Search
  • Reduz riscos jurídicos e garante Vida Útil de Projeto (VUP)

2. Simulação Visual (Veo 3.1)

  • Elabora roteiros para tours 360°
  • Cria estudos de iluminação solar
  • Gera vídeos de sequência construtiva

3. Design e Mockups (Imagen 3)

  • Gera mockups fotorrealistas de fachadas
  • Cria visualizações de interiores
  • Agiliza aprovação com clientes

4. Diagnóstico Mobile (Gemini Nano)

  • Fornece cálculos estruturais rápidos
  • Funciona em locais sem internet
  • Ideal para vistorias em campo

Como Usar?

  1. Acesse o link do agente no Gemini
  2. Descreva seu projeto com detalhes (tipo de obra, localização, dimensões)
  3. Especifique a necessidade (conformidade, cálculo, visualização)
  4. Forneça dados técnicos quando solicitado
  5. Refine os resultados através de perguntas complementares

Metodologia de Trabalho

  1. Imersão Normativa - Consulta base de conhecimento de normas
  2. Desenvolvimento no Canvas - Edição colaborativa de documentos
  3. Refinamento Multimodal - Uso de Veo, Imagen e Search
  4. Validação de Segurança - Auditoria de integridade das recomendações

30 Prompts para Engenheiros

Conformidade Normativa (1-10)

1. Analise se este projeto de edifício residencial de 8 pavimentos atende aos requisitos mínimos da NBR 15575 para desempenho acústico entre unidades.
2. Verifique se as especificações de cobrimento de armadura do meu projeto estão adequadas para Classe de Agressividade Ambiental III conforme NBR 6118.
3. Faça uma checklist de conformidade com a NBR 15575 para um projeto de casa térrea de 150m² em região de clima quente e úmido.
4. Quais são os requisitos mínimos de desempenho térmico da NBR 15575 para paredes externas na Zona Bioclimática 8?
5. Compare as exigências de vida útil de projeto (VUP) entre sistemas estruturais e sistemas de vedação segundo a NBR 15575.
6. Analise se a especificação de concreto C30 é adequada para fundações em solo com presença de sulfatos, conforme NBR 6118.
7. Quais ensaios de desempenho são obrigatórios para homologar um novo sistema de vedação vertical conforme NBR 15575?
8. Elabore um roteiro de inspeção para verificar conformidade com NBR 6118 em estruturas de concreto armado já executadas.
9. Quais são as tolerâncias dimensionais aceitáveis para elementos estruturais de concreto segundo as normas brasileiras vigentes?
10. Verifique se o projeto de impermeabilização da laje de cobertura atende à NBR 9575 e NBR 9574.

Cálculos e Dimensionamento (11-20)

11. Calcule a carga por metro linear para uma viga de baldrame com 6 metros de vão, considerando alvenaria de vedação de 19cm de espessura.
12. Qual a armadura mínima necessária para uma laje maciça de 12cm de espessura com vão de 4x5 metros?
13. Faça o pré-dimensionamento de pilares para um edifício de 10 pavimentos com área de influência de 25m² por pilar.
14. Calcule o volume de concreto necessário para uma fundação em radier de 200m² com espessura de 30cm.
15. Qual o consumo estimado de aço CA-50 para uma estrutura de concreto armado de 500m² por pavimento?
16. Dimensione a seção de uma viga contínua de dois vãos de 5 metros cada, com carga distribuída de 15 kN/m.
17. Calcule a taxa de armadura necessária para um pilar de 20x50cm com carga axial de 800 kN.
18. Qual a flecha máxima admissível para uma laje de 6 metros de vão segundo a NBR 6118?
19. Faça a verificação de punção para uma laje lisa de 20cm apoiada em pilar de 30x30cm com carga de 400 kN.
20. Calcule o espaçamento máximo entre estribos para uma viga de 20x60cm com cortante de 150 kN.

Visualização e Apresentação (21-25)

21. Crie um roteiro detalhado para vídeo de tour virtual 360° de um apartamento de 80m² com acabamento contemporâneo.
22. Elabore um prompt para o Imagen 3 gerar uma fachada moderna de edifício residencial de 12 pavimentos com elementos de concreto aparente.
23. Descreva uma sequência construtiva visual para apresentar ao cliente as etapas de construção de uma casa de 200m².
24. Crie um estudo de iluminação natural para um escritório de 100m² com fachada orientada para oeste.
25. Elabore uma apresentação visual comparativa entre três opções de revestimento de fachada para um edifício comercial.

Diagnóstico e Consultoria (26-30)

26. Analise as possíveis causas de fissuras em formato de mapa em uma laje de cobertura executada há 2 anos.
27. Quais são os critérios técnicos para decidir entre fundação superficial e profunda em um terreno com SPT médio de 8 golpes?
28. Elabore um parecer técnico sobre a viabilidade de adicionar mais um pavimento em uma edificação existente de 3 andares.
29. Quais investigações geotécnicas são necessárias para um projeto de edifício de 15 pavimentos em área urbana?
30. Desenvolva um plano de manutenção preventiva para os sistemas estruturais de um edifício residencial de 20 anos.
